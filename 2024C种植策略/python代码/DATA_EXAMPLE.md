# 2024C种植策略 - 数据文件示例

## 数据文件位置

请将以下数据文件放在 `python代码` 目录下：

```
python代码/
├── 附件2.2.xlsx
├── 附件2(1)1.xlsx
└── 聚类.xls
```

## 数据文件格式说明

### 1. 附件2.2.xlsx

**用途**: 作物种植成本和利润分析

**必需的列**:
- 作物编号 (int)
- 作物名称 (string)
- 种植成本/(元/亩) (float)
- 亩产量/斤 (float)
- 销售单价/(元/斤) (string, 格式: "最小值-最大值" 或 float)
- 地块类型 (string, 可选)

**示例数据**:
```
作物编号	作物名称	种植成本/(元/亩)	亩产量/斤	销售单价/(元/斤)	地块类型
1	小麦	500	300	2.50-4.00	水田
2	玉米	600	400	6.50-8.50	旱地
3	大豆	450	250	7.50-9.00	水田
```

### 2. 附件2(1)1.xlsx

**用途**: 地块信息和种植面积分配

**必需的列**:
- 地块编号 (int)
- 地块面积/亩 (float)
- 作物编号 (int)
- 种植面积/亩 (float)

**示例数据**:
```
地块编号	地块面积/亩	作物编号	种植面积/亩
1	10.5	1	5.0
2	12.0	2	8.0
3	8.5	1	3.5
```

### 3. 聚类.xls

**用途**: 聚类分析数据

**必需的列**:
- 作物编号 (int)
- 作物名称 (string)
- 种植成本/(元/亩) (float)
- 销售量/斤 (float)
- 平均售价 (float)

**示例数据**:
```
作物编号	作物名称	种植成本/(元/亩)	销售量/斤	平均售价
1	小麦	500	1000	3.00
2	玉米	600	1500	7.00
3	大豆	450	800	8.00
```

## 数据文件下载

这些数据文件应该来自 CUMCM 2024 竞赛官方提供的附件 C。请从竞赛官网或相关资源获取正确的数据文件。

## 数据验证

运行代码前，请确保：
1. 所有数据文件都已正确放置在代码目录下
2. 数据文件格式正确，包含所有必需的列
3. 数据文件编码为 UTF-8 或 Excel 默认编码

## 常见问题

### 1. 文件未找到错误
```
FileNotFoundError: [Errno 2] No such file or directory: '附件2.2.xlsx'
```
**解决方案**: 确保数据文件已放在代码目录下

### 2. 列名错误
```
KeyError: '种植成本/(元/亩)'
```
**解决方案**: 检查数据文件的列名是否与代码中要求的一致

### 3. 数据格式错误
```
ValueError: could not convert string to float: '2.50-4.00'
```
**解决方案**: 确保价格区间格式正确，使用"最小值-最大值"格式
