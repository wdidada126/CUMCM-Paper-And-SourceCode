# 数据文件说明

## 必需的数据文件

请将以下数据文件放在 `python代码` 目录下：

### 1. 附件2.2.xlsx
**用途**: 作物种植数据  
**包含字段**:
- 作物编号
- 作物名称
- 种植成本/(元/亩)
- 销售量/斤
- 平均售价
- 其他相关字段

### 2. 附件2(1)1.xlsx
**用途**: 地块信息数据  
**包含字段**:
- 地块编号
- 地块面积/亩
- 地块类型
- 其他相关字段

### 3. 聚类.xls
**用途**: 聚类分析数据  
**包含字段**:
- 作物编号
- 作物名称
- 种植成本/(元/亩)
- 销售量/斤
- 平均售价
- 其他相关字段

## 数据文件获取

数据文件应从全国大学生数学建模竞赛官网或竞赛题目附件中获取。

## 文件格式要求

- Excel文件格式: .xlsx 或 .xls
- 编码: UTF-8
- 第一行应为列名
- 不要删除或重命名关键列

## 数据文件检查


## 数据文件示例

如果缺少数据文件，可以创建示例数据进行测试：

```python
import pandas as pd

# 创建示例作物数据
crops_data = {
    '作物编号': range(1, 43),
    '作物名称': [f'作物{i}' for i in range(1, 43)],
    '种植成本/(元/亩)': [1000 + i*100 for i in range(1, 43)],
    '销售量/斤': [500 + i*50 for i in range(1, 43)],
    '平均售价': [5 + i*0.1 for i in range(1, 43)]
}

# 创建示例地块数据
plots_data = {
    '地块编号': range(1, 56),
    '地块面积/亩': [100 + i*10 for i in range(1, 56)],
    '地块类型': ['类型A'] * 20 + ['类型B'] * 20 + ['类型C'] * 15
}

# 保存为Excel文件
pd.DataFrame(crops_data).to_excel('附件2.2.xlsx', index=False)
pd.DataFrame(plots_data).to_excel('附件2(1)1.xlsx', index=False)
pd.DataFrame(crops_data).to_excel('聚类.xls', index=False)
```

## 常见问题

### Q: 数据文件路径错误？
A: 确保数据文件与Python代码在同一目录下，或者修改代码中的文件路径。

### Q: 数据文件格式不支持？
A: 确保使用Excel格式（.xlsx 或 .xls），不要使用.csv或其他格式。

### Q: 数据文件列名不匹配？
A: 检查数据文件的列名是否与代码中要求的列名一致，必要时修改数据文件或代码。

## 技术支持

如有问题，请检查：
1. Python版本是否为3.7或更高
2. 是否已安装所有依赖包
3. 数据文件是否正确放置在代码目录下
4. 数据文件格式是否正确
