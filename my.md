# 2. 一段最简单、最标准的 LINGO 代码长啥样？
（直接看一眼就懂）

```lingo
! 最简单的线性规划模型;
MODEL:
MAX = 2*X + 3*Y;  ! 目标：最大化利润;

X + Y <= 10;      ! 约束1;
2*X + Y <= 15;    ! 约束2;

X >= 0;
Y >= 0;
END
```

运行后直接输出：
- 最大利润
- X、Y 最优取值

**这就是 LINGO 代码**
