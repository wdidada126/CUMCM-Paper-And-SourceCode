# 数据文件说明

## 输出文件说明

本项目运行后会生成以下Excel和图片文件：

### 1. decision_results.xlsx
**用途**: 问题2的决策分析结果  
**包含内容**:
- 6种不同情况的决策分析结果
- 每种情况下的最优决策组合
- 各种决策方案的利润、收入和成本

### 2. multi_stage_production_results.xlsx
**用途**: 问题3的多阶段生产决策结果  
**包含内容**:
- 所有可能的决策组合
- 每种决策的利润、收入和成本
- 最优决策方案

### 3. defect_rate_distribution.png
**用途**: 次品率估计分布图  
**包含内容**:
- 50000次模拟的次品率估计分布
- 真实次品率和平均估计次品率的对比
- 统计指标（均值、标准差、置信区间等）

### 4. optimal_strategies_distribution.xlsx
**用途**: 最优策略分布统计  
**包含内容**:
- 不同最优策略的出现次数
- 最常见的最优策略

## 数据文件获取

这些文件是在运行代码后自动生成的，无需预先准备。

## 文件格式要求

- Excel文件格式: .xlsx
- 图片格式: .png
- 编码: UTF-8

## 常见问题

### Q: 数据文件不存在？
A: 这些文件是在运行代码后自动生成的。请先运行相应的模块。

### Q: 数据文件路径错误？
A: 确保在代码目录下运行代码，或者修改代码中的文件路径。

### Q: 数据文件格式不支持？
A: 确保使用Excel格式（.xlsx），不要使用.csv或其他格式。

## 技术支持

如有问题，请检查：
1. Python版本是否为3.7或更高
2. 是否已安装所有依赖包
3. 是否有足够的计算资源运行决策分析
4. 代码中的文件路径是否正确
